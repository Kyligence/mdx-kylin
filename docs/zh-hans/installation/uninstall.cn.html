卸载

卸载

在本节中，我们将为您介绍卸载 MDX for Kylin 的步骤。

  • 在所有 MDX for Kylin 节点上运行下述命令以停止 MDX for Kylin 实例：

     bin/mdx.sh stop

  • 在所有 MDX for Kylin 节点上运行下述命令以删除工作目录：

     rm -rf <MDX_DIR>

  • 至此，MDX for Kylin 卸载完成。
